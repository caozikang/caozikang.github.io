通过命令行关闭WAS应用

2020-05-27

WAS Script script/appstop.sh内容如下： #‌!/bin/bash cd /data/opt/IBM/WebSphere/AppServer/profiles/Dmgr01/bin ./wsadmin.sh -user wasadmin -password password -lang jython -f appstop.py exec bash appstop.py内容如下（位于： /data/opt/IBM/WebSphere/AppServer/profiles/Dmgr01/bin路径下） #‌Starting the Registry Services application app_name = ‘Help’ #‌Identify the application manager MBean for the server where the application resides, and assign it to the appManager variable appManager = AdminControl.queryNames(‘cell=oadmgrp01Cell01,node=oaconnp01Node01,type=ApplicationManager,process=InfraCluster_server1,*') print appManager #‌Starting the application now AdminControl.invoke(appManager, ‘stopApplication’, app_name)
